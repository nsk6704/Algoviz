AlgoViz is an interactive web-based platform designed to visualize algorithms in a clear, step-by-step manner. It transforms abstract algorithmic concepts into dynamic visual processes, allowing users to observe how data evolves and how decisions are made during execution. By focusing on real-time feedback and controlled execution, the platform helps build intuitive understanding rather than relying solely on theoretical explanations.

The current implementation focuses on selected concepts from machine learning and artificial intelligence, including K-Means clustering, K-Nearest Neighbors, Logistic Regression, Naive Bayes, A* search, Minimax with alpha-beta pruning, and Hill Climbing. It is intended as a practical learning tool that makes complex algorithms more accessible through interactive visualization.
